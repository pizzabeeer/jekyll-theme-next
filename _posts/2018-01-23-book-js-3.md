# 함수<br/>

```javascript
// sayHello라는 함수를 선언
function sayHello() {
	//함수 바디는 여는 중괄호로 시작하고
    console.log("hello World");
    //닫는 중괄호로 끝난다.
}

// 함수를 호출
sayHello(); //hello World
```

## 반환값<br/>
- return 키워드를 사용하면 함수를 종료하고 값을 반환한다.<br/>
- return을 명시적으로 호출하지 않으면 반환 값은 undefined가 된다.

```javascript
function sayHello() {
	return "heelo World";
}

sayHello(); //hello World
```
